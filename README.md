# vim_config pathogen  
a simple vim config by pathogen  
usage:  
1.将.vim文件夹和.vimrc配置文件复制到/home路径下  
2.下载所需插件保存到.vim/bundle文件夹中  
3.在.vimrc设置插件相关属性  
项目中保存并设置了常用的vim插件，个人可根据自己的情况自由配置。
